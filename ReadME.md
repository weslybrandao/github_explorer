## 📁 Download

```bash
# Cloning repository
$ git clone https://github.com/weslybradao/github_explorer

# Entering the project folder
$ cd github_explorer

# Installing the dependencies
$ yarn install or npm install

# Starting the application
$ yarn start or npm start


```
